## Acceptance Notes

This batch is accepted as a coherent visual package, not yet wired into the
game. It captures the desired third-set direction:

- brighter rural World War I rear-area environment
- nature-forward fields, linen, farm structures, mud, rail siding, and practical
  care
- no active combat or battlefield spectacle
- ordinary usefulness beside implied mass injury
- seven images that read as one anti-story without becoming one continuous
  camera move

Future implementation should treat these as source/reference assets until a
real background-selection pipeline exists.
