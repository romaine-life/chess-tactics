# Font Licenses

This directory contains only the license and attribution text for font binaries
stored behind the backend live-media catalog. No font bytes or runtime font
pointers are served from this repository path.

- `jersey-10`: Jersey 10, SIL Open Font License 1.1
- `press-start-2p`: Press Start 2P, SIL Open Font License 1.1
- `vt323`: VT323, SIL Open Font License 1.1
- `newsreader`: Newsreader, SIL Open Font License 1.1
- `advance-wars-2-gba`: Advance Wars 2 GBA by Patrick H. Lauke, Creative
  Commons Attribution 3.0

The live `.woff2` candidates use the same browser subsets formerly sourced from
Google Fonts for the families and unicode ranges declared by the UI. Each family
directory keeps its upstream `OFL.txt` as text provenance.

The live Advance Wars 2 GBA `.otf` candidate retains its original FontStruct
`readme.txt` and `license.txt` attribution here.
