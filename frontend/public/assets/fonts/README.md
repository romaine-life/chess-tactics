# Vendored Fonts

These fonts are served from the app so accepted UI typography does not depend on
Google Fonts at runtime.

- `jersey-10`: Jersey 10, SIL Open Font License 1.1
- `press-start-2p`: Press Start 2P, SIL Open Font License 1.1
- `vt323`: VT323, SIL Open Font License 1.1
- `newsreader`: Newsreader, SIL Open Font License 1.1

The `.woff2` files are the same browser subsets served by Google Fonts for the
families and unicode ranges declared in `frontend/src/style.css`. Each family
directory includes its upstream `OFL.txt`.
